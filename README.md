## OCR WEB SERVICE
Before running the OCR web service, you need to run Minio and PostgreSQL first.

### MINIO
1. go to docker folder
2. run `docker-compose up -d`

### POSTGRESQL
Migracja:
1. Run `dotnet ef database update` (must install dotnet-ef first)
